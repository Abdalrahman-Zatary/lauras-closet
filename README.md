# Laura's Closet
A single-page landing website for *Laura's Closet*, a simple clothing store showcasing collections, brands, promotions, and service features, built with HTML, CSS, JavaScript, and Bootstrap 5.

## Screenshot
![Laura's Closet Screenshot](./assets/images/screenshot.jpg)

## Features
- Display clothing collections and brands
- Promotions/Offers section
- Interactive product cards
- Fixed navigation with animation scroll
- Fully responsive layout for mobile and desktop
- Built with Bootstrap 5 for rapid design and custom CSS for styling
- Simple JavaScript interactivity (e.g., modal, offer highlights)

## Built With
- HTML5
- CSS3
- JavaScript (vanilla)
- Bootstrap 5 (Local integration)

## What I Learned
- Integrating Bootstrap 5 with custom CSS
- DOM manipulation and event handling in JavaScript
- Organizing a single-page layout with multiple sections
- Responsive design and mobile-first adjustments
- Structuring project files for better maintainability
